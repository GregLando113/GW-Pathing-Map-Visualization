### Usage ###
Launch Loader.exe or Loader.au3
(optional) search for map name
double click on a map
map will open
mouse drag to move around, wheel to zoom, spacebar to toggle wireframe, c to toggle ranges

### Advanced Usage ###
you can launch Visualization.exe directly by command line, 1st parameter is mapfile ID
